WebSocket idea

WebSocket keeps the connection open.

HTTP = Phone call where you hang up after every sentence

WebSocket URL formats
ws:// → normal WebSocket (like HTTP)

wss:// → secure WebSocket (like HTTPS)
The server sends a message to multiple connected clients.=broadcast
Client connects → socket connection

Client sends message → socket.emit()

Server receives message → socket.on()

Server sends to others → broadcast
