# One_vote_per_user_app

This is a voting application. You can create polls, people can vote on them, and you can see the results.

Login & Register: Create an account to start voting.
Admin: Admin can create poll!
Create Polls: Ask a question and add as many options as you want.
Vote Safely: The app makes sure every user only votes once per poll.
Live Results: See the percentage (%) of votes for each option immediately after you vote.

# Start the Backend

Open a terminal (CMD).
Go to the backend folder.
Type npm install.
Type npm run dev.

# Start the Frontend

Open a new terminal.
Go to the frontend folder.
Type npm install.
Type npm run dev.
Open your browser and go to: `http://localhost:5173`

# How to use

Register: Go to the Register page and create an account.
Login: Sign in with your email and password.
Create: Click the blue "Create Poll" button at the top to make your first poll.
Vote: Go to the Home Page, click "Vote Now" on any poll, pick an option, and click "Cast My Vote".
Results: Once you vote, you will see blue bars showing the percentages.

# Technology used

Backend: Node.js & Express
Database: MongoDB
Frontend: React.js
Design: CSS
