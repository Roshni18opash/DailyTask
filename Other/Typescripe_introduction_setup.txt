Day 4 – Introduction to TypeScript
 Learning:
o What is TypeScript 
 TypeScript is JavaScript with added syntax for types.
TypeScript is JavaScript with rules that help you write safer and cleaner code.

why we use it?
✔ Finds errors before running the code
✔ Makes code easier to understand
✔ Helps when working on big projects
✔ Used by many companies (Google, Microsoft, etc.
JavaScript lets you do anything.
TypeScript helps you do it correctly.

Browsers do not run TypeScript directly.
TypeScript is converted into JavaScript, then the browser runs it.
