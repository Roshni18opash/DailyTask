 Interfaces & typr aliases:

1.What is an Interface?
An interface is used to define the structure (shape) of an object.
example:
interface User {
  id: number;
  name: string;
  isActive: boolean;
}
const user1: User = {
  id: 1,
  name: "Roshni",
  isActive: true
};
 Used mainly for objects
Very common in APIs & React props

 When to Use What?
 Use Interface when:
Object shape define karni ho
React props / state
Public API models
Class implement karna ho

Use Type Alias when:
Union / intersection chahiye
Function types
Complex types
Utility types ke saath kaam

Interface is best for object structure and extensibility, Type alias is best for flexibility and complex types.

 What is a Typed Object?
A typed object is an object whose structure is enforced by a type or interface.
